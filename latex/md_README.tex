This repository contains code for an assignment focused on expanding your documentation toolkit using Doxygen. The assignment involves documenting multiple classes representing employees in a company hierarchy.\hypertarget{md_README_autotoc_md1}{}\doxysection{Assignment Overview}\label{md_README_autotoc_md1}
The assignment tasks include\+:
\begin{DoxyEnumerate}
\item Creating Doxygen configuration files.
\item Documenting the provided \mbox{\hyperlink{classEmployee}{Employee}}, \mbox{\hyperlink{classOfficer}{Officer}}, and \mbox{\hyperlink{classSupervisor}{Supervisor}} classes.
\item Generating H\+T\+ML documentation using Doxygen.
\end{DoxyEnumerate}\hypertarget{md_README_autotoc_md2}{}\doxysection{Assignment Components}\label{md_README_autotoc_md2}

\item {\bfseries{\mbox{\hyperlink{Officer_8h}{Officer.\+h}}, \mbox{\hyperlink{Officer_8cpp}{Officer.\+cpp}}}}\+: Files containing the declaration and implementation of the \mbox{\hyperlink{classOfficer}{Officer}} class, which inherits from the \mbox{\hyperlink{classEmployee}{Employee}} class.
